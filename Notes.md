# Type of css

1. Inline Css

```
 <p style="background-color: aqua;">Lorem ipsum dolor </p>
```
2. Internal Css

```
<head> 
    <style>
        p{background-color:blueviolet }
        h1{color: green};
    </style>
</head>
```

3. External Css
    * Make a Css File
``` 
h1{
    font-size: medium;
    text-align: center;
    }
```
